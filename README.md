<h1 align="center">Welcome to Image Filter 👋</h1>
<p>📸  Program to apply image filters in .bmp images</p>

## Install

```sh
git clone https://github.com/peguimasid/Image-Filter.git
```

## Compile

```sh
gcc *.c -o filter
```

## Run

```sh
./filter -r IMAGE.bmp REFLECTED.bmp
```

### Options:

1. -b ➡ blur
2. -g ➡ grayscale
3. -r ➡ reflect
4. -s ➡ sepia
